The Same Data as FlightRadar24
ADS-B (Automatic Dependent Surveillance–Broadcast) transponders broadcast GPS position, altitude, speed, and identity every ~0.5s on 1090MHz. FlightRadar24, Flightaware, and SENTINEL all decode the same raw signal from ~40,000 ground receivers worldwide. OpenSky Network is the open-access nonprofit version. SENTINEL polls every 15 seconds — real callsigns, real positions.
aisstream.io — Real Vessel Tracking
AIS (Automatic Identification System) is mandatory for vessels >300 GT. Register free at aisstream.io for a WebSocket key. Provides real-time vessel positions, headings, and identities — same data as MarineTraffic and VesselFinder.
GNSS Jamming via ADS-B Anomalies
GPSJam.org detects interference by analyzing NACp (Navigation Accuracy Category) values in ADS-B messages. When aircraft report degraded GPS, those anomalies are aggregated geospatially. The 40,000-node ADS-B network becomes a passive GNSS sensor array — no dedicated hardware needed.
